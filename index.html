Tugas 1

Konsep Dasar Algoritma

Pemahaman algoritma sehari-hari dan struktur dasar pemrograman.

1. Algoritma di Kehidupan Sehari-hari

a. Mengambil uang di ATM:

  1. Masukkan kartu ATM ke dalam mesin.
  2. Pilih bahasa yang ingin digunakan.
  3. Masukkan 6 digit PIN ATM.
  4. Pilih menu "Tarik Tunai" atau pilih nominal yang tertera.
  5. Ambil uang yang keluar dari mesin.
  6. Ambil kembali kartu ATM.

b. Membuat kopi yang rasanya pas:

  1. Rebus air hingga mendidih.
  2. Siapkan cangkir kosong.
  3. Masukkan 1 sendok teh bubuk kopi dan 1.5 sendok teh gula ke dalam cangkir.
  4. Tuangkan air panas secukupnya.
  5. Aduk perlahan hingga rata.
  6. Cicipi, jika kurang manis tambah gula sedikit, jika sudah pas kopi siap dinikmati.

c. Menghitung nilai y dari persamaan y = 3x + 6:

  1. Mulai program.
  2. Tentukan atau masukkan nilai x.
  3. Kalikan nilai x dengan 3.
  4. Tambahkan hasil perkalian tersebut dengan 6.
  5. Simpan hasilnya ke dalam variabel y.
  6. Tampilkan nilai y.

2. Apa itu Algoritma dan Program?

  • Algoritma: Urutan langkah-langkah logis dan sistematis yang disusun untuk menyelesaikan suatu masalah tertentu.
  • Program: Kumpulan instruksi yang ditulis menggunakan bahasa pemrograman (berdasarkan algoritma) agar dapat dieksekusi oleh komputer.

3. Tiga Struktur Dasar Algoritma

  • Sekuensial (Sequential): Struktur di mana instruksi dieksekusi secara berurutan baris demi baris, dari atas ke bawah tanpa adanya loncatan.
  • Pemilihan (Selection): Struktur percabangan di mana alur program akan memilih langkah berikutnya berdasarkan kondisi tertentu (TRUE/FALSE). Contoh: IF - ELSE.
  • Perulangan (Repetition): Struktur yang memungkinkan satu atau serangkaian instruksi dieksekusi berkali-kali selama kondisi tertentu terpenuhi. Contoh: FOR, WHILE.

4. Fungsi Bahasa Pemrograman

Berfungsi sebagai jembatan komunikasi antara manusia dan komputer. Bahasa pemrograman menerjemahkan logika manusia (algoritma) ke dalam bahasa mesin agar komputer dapat melakukan tugas tertentu, seperti membuat aplikasi, mengolah data, hingga mengontrol perangkat keras.

Tugas 2

Algoritma Sekuensial

Kasus: Menghitung Luas Lingkaran.

1. Deskripsi / Flowchart (Teks)

  • START
  • INPUT: Masukkan nilai jari-jari (r).
  • PROCESS: Hitung Luas = 3.14 * r * r.
  • OUTPUT: Tampilkan Luas.
  • END

2. Pseudocode


ALGORITMA HitungLuasLingkaran
DEKLARASI
    r : float
    luas : float
    phi : float = 3.14
DESKRIPSI
    Baca(r)
    luas <- phi * r * r
    Tulis(luas)

3. Program (Python)


# Program Sekuensial Menghitung Luas Lingkaran
phi = 3.14
r = float(input("Masukkan nilai jari-jari: "))

# Proses hitung
luas = phi * r * r

# Output
print(f"Luas lingkaran dengan jari-jari {r} adalah {luas}")
Tugas 3

Struktur Perulangan (Looping)

Implementasi menggunakan bahasa Python.


# ==========================================
# 1. Perulangan Struktur FOR
# ==========================================
print("--- Struktur FOR ---")
for i in range(1, 6):
    print(f"Perulangan For ke-{i}")


# ==========================================
# 2. Perulangan Struktur WHILE
# ==========================================
print("\n--- Struktur WHILE ---")
x = 1
while x <= 5:
    print(f"Perulangan While ke-{x}")
    x += 1


# ==========================================
# 3. Perulangan Struktur DO-WHILE (Simulasi di Python)
# ==========================================
print("\n--- Struktur DO-WHILE ---")
y = 1
while True:
    print(f"Perulangan Do-While ke-{y}")
    y += 1
    if y > 5:
        break


# ==========================================
# 4. Perulangan Bersarang (NESTED LOOP)
# ==========================================
print("\n--- Nested Loop (Bintang) ---")
for baris in range(1, 4):
    for kolom in range(1, baris + 1):
        print("*", end=" ")
    print() # Pindah baris
